FAQ about Swarachakra

General

1.What are the languages in which swarachakra is available?

Ans.Currently it is available in 9 languages (as shown on home page) with more languages to be added in near future.

2.What platforms does it support?

Ans. Swarachakra is available for android devices & as Webapp(for Desktop).We plan to extend it across windows and ios platforms too.

3.Where can I download Swarachakra android app?

Ans.Choose your language on home page -> Click the playstore icon on top right corner -> You will land on playstore page for swarachakra ->Click the Install button ->Follow as directed.-> Installed 😊

4.Can I type all possible words?

Ans. Yes,you can type all possible word as it is phonetic based keyboard.Please take a look at examples shown for typing unique words if you are facing some difficulty.

5.Why letters on keyboard are not visible in my phone?

Ans.Currently, some manufatures do not support some of the Indian languages like Gujarati,Punjabi and Oriya.Check it with the phone guidelines provided by your manufacturers about your language support.

6.Why it occupies so much screen?

Ans.Since Indian Languages have a more number of letter,it becomes inevitable to use more screen size.But please check out the latest version of your Swarachakra App,we have tried to address this issue effectively.

7.Why Swarachakra doesn't predict like other keyboard?

Ans.Please checkout the latest version.Currently,Text prediction is supported for Hindi with more languages to be added soon.

4.How to type on Swarachakra?

Ans.Check the Swarachakra page of your language.In the bottom, you will find 'How to Install' and 'How to Type video'.Please watch it.

4.Can we change the size of Chakra?

Ans.No,the size is fixed.But,even if you slide outside the chakra in the required angle,it will work.

4.Can i type English with Swarachakra?

Ans. Yes ,you can type.

We are happy to help if you have any further queries.Please post your problems on or facebook page or as comment in the dialog box.
